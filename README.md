# CareConnect 🚑

A HealthTech application designed to enhance security surveillance and detect suspicious activities in real-time.

## 🚀 Features
- Real-time monitoring
- Suspicious activity detection
- Alert system

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Python (FastAPI)

## 📂 Project Structure
- /frontend → React app
- /backend → API & logic

## 🎯 Goal
To build a smart surveillance system that improves safety using AI.
